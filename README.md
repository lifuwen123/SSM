# SSM
## 一个由bootstrap搭建的ssm案列
我更新了
技术选用:spring+springmvc+mybatis+bootstrap+redis等  
功能清单:前台用户登录、记住用户名密码、找回密码、后台有模糊查询系统、修改密码、分页展示图片、投诉某个用户、图片下载、轮播图效果    
管理员后台可以对用户实现CRUD功能、批量操作用户、查看用户的登录记录、日志记录功能、进入反馈系统等
